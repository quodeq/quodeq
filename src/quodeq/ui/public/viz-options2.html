More Visualization Options

Interactive demos with mock codebase data. Hover for details.

1

Icicle Chart — Horizontal Hierarchy

Each row = one folder depth. Width = violation count. Color = severity. Click to zoom into a subtree. Very readable for deep hierarchies. Like a horizontal treemap but with clear parent-child rows.

2

Scatter Plot — Violations vs Compliance

X = violations, Y = compliance. Dot size = total findings, color = worst severity. Instantly spot outliers: bottom-right = bad (many violations, little compliance). Top-left = good. Hover for file details.

3

Flame Chart — Vertical Stack

Root at top, deepest files at bottom. Width = violation count. Familiar from CPU profilers. Click to zoom. Color = severity. Great for identifying deep hotspots.

4

Voronoi Diagram — Organic Regions

Each polygon = one file. Area approximates violation count. Color = severity. Organic, map-like feel. Hover highlights region. Good for seeing relative size of problem areas.
